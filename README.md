# Interfaz Web para Consultas SQL

Mini proyecto desarrollado con Flask como parte de un curso. Permite ingresar consultas SQL y mostrar los resultados en tablas desde una interfaz web. 

Aunque fue implementado usando una base de datos de aviones, la lógica puede adaptarse a otros conjuntos de datos.

## Tecnologías
- TypeScript
- JavaScript
- Python
- Flask
- HTML
- CSS
- SQL
